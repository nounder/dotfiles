This is the pi configuration directory.

You can find local source in `./repo`:

- `packages/agent` (`environment-agent-core`) — General-purpose agent with transport abstraction, state management, and attachment support
- `packages/ai` (`environment-ai`) — Unified LLM API with automatic model discovery and provider configuration
- `packages/coding-agent` (`environment-coding-agent`) — Coding agent CLI with read, bash, edit, write tools and session management
- `packages/mom` (`environment-mom`) — Slack bot that delegates messages to the environment coding agent
- `packages/pods` (`environment`) — CLI tool for managing vLLM deployments on GPU pods
- `packages/tui` (`environment-tui`) — Terminal User Interface library with differential rendering for efficient text-based applications
- `packages/web-ui` (`environment-web-ui`) — Reusable web UI components for AI chat interfaces powered by environment-ai
